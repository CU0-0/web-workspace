document.write('출력하고자하는 문구')

요소객체.innerHTML / innerText = '요소에 출력할 내용'

innerHTML / innerText
요소(Element)의 값(Content)를 조작할 수 있는 속성 두 가지

=> get : 요소.innerHTML / innerText
=> set : 요소.innerHTML / innerText = '대입할 값';

입력받는 방법 !!

1. window.confirm('질문내용')

        confirm() 호출 시 인자로 전달한 '질문 내용'과 확인 / 취소 버튼이 존재하는 창이 뜸
        

        확인 버튼을 클릭 시 true값 반환, 취소 버튼 클릭 시 false값 반환

수량

0개

2. window.prompt('질문내용')

        prompt() 호출 시 '질문 내용'과 사용자가 값을 입력할 수 있는
        input요소와 확인 / 취소 버튼이 보여지는 창 생성 


        - 확인 버튼 클릭 시 텍스트상자에 입력한 값을 반환 

        - 취소 버튼 클릭 시 null값을 반환
