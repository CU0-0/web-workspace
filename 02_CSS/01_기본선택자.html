선택자(Selector)란?? 중요!!

특정 HTML요소만 선택하려고할때 아용하는 문법
선택자를 이용하면 원하는 요소만 쏙쏙 골라서 style속성을 부여할 수 있음

잘 안쓰는 방식

1. 전체(모든 요소) 선택자

        *{
            스타일 속성 : 속성값;
            스타일 속성 : 속성값;
        }

        => 현재 이 HTML분서 내의 모든 요소를 다 선택해서 스타일을 부여

2. 태그 선택자

        태그명 {
            스타일 속성 : 속성값;
            스타일 속성 : 속성값;
        }

        => 문서 내의 같은 태그들을 전부 선택할 때 사용

3. 아이디 선택자

        #아이디속성값{
            스타일 속성 : 속성값;
            스타일 속성 : 속성값;
        }

        => 문서 내의 특정 HTML요소 단 하나만을 선택할 때 사용
        단, 해당 요소에 반드시 id속성값이 존재해야만하며 다른 요소와 중복되면 안됨
        id값은 반드시 하나의 문서에서 고유한 값이어야함 !!!!
        동일한 태그더라도 부분적으로 다르게 스타일을 부여하고 싶을 때 사용
  1. 아이디선택자
  2. 아이디선택자
  3. 아이디선택자
  4. 아이디선택자

4. 클래스 선택자

        .클래스속성값{
            스타일 속성 : 속성값;
            스타일 속성 : 속성값;
        }

        => 현재 HTML문서에서 여러 개의 요소를 선택할 때 사용
