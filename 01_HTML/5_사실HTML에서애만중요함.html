form태그 & submit 버튼

from action="" - 폼태그 작성시 필수 속!! method="" - 꼭 써주는게 좋음

* action : form안에 사용자가 입력한 값들을 전달하면서 요청을 보낼 URL 을 작성
* method : 요청 전송방식을 지정한 form태그 내부에 존재하는 submit버튼을 클릭 시
action속성에 작성한 서버의 경로로
method속성에 작성한 요청 방식을 통해
사용자가 input요소에 입력한 값들을key=value형태로 전달
